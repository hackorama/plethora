Plethora Configuration Properties

These are defined in the configuration file provided as an argument to Plethora startup and follows the well known simple java properties file format

NameDefault ValueTypeDescription
web.root web Optional The location of the static web content, defaults to location relative to install root
log.root log Optional Log file location, defaults to location relative to install root
data.refresh.seconds 5 Optional Defined in seconds, how often metric data is collected from the application modules
connection.retry.seconds 60 Optional Defined in seconds, How often to poll for application module connections
http.server.thread.pool.size 2 Optional Thread pool size for HTTP metric server
snmp.server.thread.pool.size 2 Optional Thread pool size for SNMP metric server
system.process.rescan.secs 600 Optional Defined in seconds, how often to rescan for application process ids, in case they have restarted with a new pid
system.data.caching.secs system.data.caching.secs Optional Defined in seconds, how long to cache the system metrics collected. Higher value reduces any system metric collection overhead
system.metrics true Optional Enables or disables system metrics collection
